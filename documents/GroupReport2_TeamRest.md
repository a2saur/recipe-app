## 1\. How you used Generative AI

*Describe specific tasks where you used AI tools (e.g., code generation, debugging, polishing UI, testing).*  
Generative AI aided in creating some test cases for iteration 3 with the code completion feature to help generate assertion statements.

*Which tasks did it help you the most?*  
It helped most when there were a lot of assertion statements needed for a test block. It was also helpful in creating get and set statements especially when the database is this large it becomes hard to remember all of the required fields.

## 2\. Impact on your development process

*How did AI tools influence your productivity and collaboration?*  
The use of AI increased our productivity, and we were able to get more done in iteration 3 than seemed possible because of how expansive the test cases had to be.

*Did AI change how your team divided responsibilities?*  
AI did not change how we divided responsibilities, as we divided the test cases based on the routes between the four of us. Without AI, we might have divided it up more since some of the route files had more than others.

*Did AI save time, create additional work, or shape design decisions?*  
AI saved time in the final iteration, it made it easier to cover more of the use cases and it got easier to use overtime. When the test cases were similar it was much faster at generating the correct line of code. However, it was a small portion of the work we had to get done, so AI didn’t save a significant amount of time or shape design decisions.

## 3\. Evaluation of benefits and drawbacks

*What were the most useful aspects of using AI?*  
The most useful aspect of using AI was its speed, as it was able to generate statements quickly and it allowed less going back and forth between the files.

*What challenges, inaccuracies, or limitations did you encounter?*  
There were some times when the code generated was not correct and it became harder to debug than it would be writing the statement from scratch. For example, when updating the tags or recipes in a cookbook it did not do as well with the lists.

*Did you need to correct or verify AI-generated content?*  
Any AI-generated content needed to be verified, especially since there were times when the generated code was not correct. In most cases it had to do with incorrect form like an underscore instead of period.

## 4\. Your overall perspective

*Do you feel the use of AI tools enhanced or hindered your learning?*  
The use of AI did not hinder learning, considering we only used it for our test cases. We learned enough about it in class paired with the quiz that it was able to expedite our time creating test cases in iteration 3\. This allowed us to focus more on error fixing and other features such as sso login.

*What guidelines would you recommend for future students using AI in software engineering projects?*  
For future students, we’d recommend making sure to understand the code that is being generated so that if there are any problems that occur it is easy to debug.
